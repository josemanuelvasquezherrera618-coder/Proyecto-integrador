## Nombre del Proyecto
Sistema de Gestión Educativa - Fase 1

## Descripción
Este proyecto es una aplicación de gestión educativa desarrollada en Java. Permite a los usuarios gestionar información relacionada con estudiantes, profesores, cursos y calificaciones. La aplicación está diseñada para ser fácil de usar y proporciona una interfaz de línea de comandos para interactuar con el sistema, Además este repositorio contiene la primera fase del proyecto integrador para la asignatura FrontEnd 1.

## Intrucciones para el Uso
1. Clonar el repositorio en tu máquina local.
2. Navegar al directorio del proyecto.
3. Importante: Abra la consola del navegador (F12) para visualizar la impresión de datos solicitada
4. Seguir las instrucciones en la consola para interactuar con el sistema.
5. Para salir de la aplicación, ingresa la opción correspondiente en el menú principal.
6. Asegúrate de tener Java instalado en tu máquina para compilar y ejecutar la aplicación.
7. Puedes agregar datos de estudiantes, profesores, cursos y calificaciones a través de la interfaz de línea de comandos. La aplicación también permite visualizar la información almacenada en el sistema.
8. Recuerda que esta es solo la primera fase del proyecto, por lo que se espera que se agreguen más funcionalidades y mejoras en futuras fases.
